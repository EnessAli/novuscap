# USER STORIES

## Kullanıcı Hikayeleri

### Admin
- "Bir admin olarak, yeni bir startup ekleyebilmek istiyorum ki harita üzerinde temsil edilsin."
- "Bir admin olarak, kullanıcıları yönetebilmek istiyorum ki sistemdeki erişimlerini kontrol edebileyim."
- "Bir admin olarak, marker'ları düzenleyebilmek istiyorum ki harita üzerindeki bilgileri güncel tutabileyim."

### Editör
- "Bir editör olarak, içerik ekleyip düzenleyebilmek istiyorum ki kullanıcılar için güncel bilgiler sağlayabileyim."
- "Bir editör olarak, marker'ları gizli yapabilmek istiyorum ki sadece belirli kullanıcılar bu bilgilere erişebilsin."

### Girişim Temsilcisi
- "Bir girişim temsilcisi olarak, kendi girişimimin bilgilerini güncelleyebilmek istiyorum ki potansiyel yatırımcılar için doğru bilgiler sunabileyim."
- "Bir girişim temsilcisi olarak, girişimimin istatistiklerini görebilmek istiyorum ki performansımı değerlendirebileyim."

### Gözlemci
- "Bir gözlemci olarak, harita üzerinde yatırımcıları filtreleyebilmek istiyorum ki sadece ilgilendiğim marker'ları görebileyim."
- "Bir gözlemci olarak, girişimlerin detaylarını inceleyebilmek istiyorum ki hangi girişimlerin aktif olduğunu öğrenebileyim."
